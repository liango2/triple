# TRIPLE

How to start:

1. Start REPL, the default namespace should be ~triple.dev~.

    ```shell
    $ lein repl
    ```
2. Start the figwheel.

    ```clojure
    (start-figwheel!)
    ```
3. Switch to ClojureScript REPL.

   ```clojure
   (cljs-repl)
   ```
4. Open <http://localhost:3449> with browser.

> There's no basic code for clojure server yet.
